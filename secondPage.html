3. What is event bubble?

When an element is clicked, an event is directed to the element.
If an event handler is set for the element, the event handler is triggered.
Then the event "bubbles up" to the elements parent.

4. What is a callback function and why will you use it?

A JavaScript callback is a function which is to be executed after another function has finished execution.
